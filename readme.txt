#C-Syntax Checker

This project was created by a group of 3 Students- Bhuvnesh Jain, Lakshit Bhutani and Chirag Agarwal as a part of "Logic in Computer Science" Assignment.

For running the code on your computer, you must have "SWI Prolog" installed on it. You can get it from [here][111]. 

For refering to the code, you can look at LCS_Project.pl

For refering to input format and some information about the code, refer to Basic information.md

For refering to some test cases, you can look at testcases.md

For refering to Conversion to Reverse Polish notation of expressions, refer to appendix.md
